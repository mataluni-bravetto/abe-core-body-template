## Quick Start (4 Minutes)

### Step 1: Duplicate Template (30 seconds)
```bash
./scripts/duplicate.sh my-project
cd ../my-project
```

### Step 2: Install Dependencies (1 minute)
```bash
make install
```

### Step 3: Start Backend (30 seconds)
```bash
make dev-backend
# Backend running at http://localhost:3001
```

### Step 4: Start Frontend (30 seconds)
```bash
make dev-frontend
# Frontend running at http://localhost:3000
```

### Step 5: Verify (1 minute)
- ✅ Backend health check: http://localhost:3001/health
- ✅ Frontend rendering: http://localhost:3000
- ✅ Ready to build features

**Total Time: 4 minutes**  
**Traditional Setup: 2-3 days**

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Git

### Installation
```bash
# Clone repository
git clone https://github.com/bravetto/abe-core-body-template.git
cd abe-core-body-template

# Duplicate for new project
./scripts/duplicate.sh my-project
cd ../my-project

# Install dependencies
make install

# Start development
make dev-backend  # Terminal 1
make dev-frontend # Terminal 2
```
