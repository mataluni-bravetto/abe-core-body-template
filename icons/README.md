# Extension Icons

This directory contains the extension icons for the AiGuardian Chrome Extension.

## Required Icon Sizes
- **16x16px** - Browser toolbar and context menus
- **32x32px** - Windows taskbar and settings
- **48x48px** - Extension management page
- **128x128px** - Chrome Web Store and installation

## Icon Guidelines
- **Format**: PNG with transparent background
- **Style**: Simple, recognizable design
- **Colors**: Use your brand colors
- **Design**: Should be clear at small sizes (16px)

## Current Status
⚠️ **Placeholder icons needed** - Replace these with your actual extension icons.

## Creating Icons
1. Design your icon at 128x128px
2. Scale down to other sizes maintaining clarity
3. Save as PNG with transparent background
4. Replace the placeholder files in this directory

## Tools for Icon Creation
- **Figma** - Free online design tool
- **GIMP** - Free image editor
- **Adobe Illustrator** - Professional vector graphics
- **Canva** - Simple online design tool

## File Naming
- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels  
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels
